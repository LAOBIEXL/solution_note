如果此时b和r能够被另一个数整除，那么a也可以被这个数整除，即：
其实有

==*除数和被除数的最大公因数  =   除数和余数的最大公因数*==

```cpp
#include<iostream>
using namespace std;

int main()
{
	int m, n;
	cout << "请输入两个正整数：";
	cin >> m >> n;

	int a = m, b = n;
	//求最大公因数：
	while (a % b != 0)
	{
		int t = a % b;
		a = b;
		b = t;
	}
	cout << "最大公因数是 ：" << b << endl;

	//求最小公倍数：
	//由两个数的乘积等于最大公约数乘以最小公倍数：

	cout << "最小公倍数是 ：" << (m * n / b) << endl;

	return 0;
}
```
