对于一般的归并排序，一般采用创建辅助数组的方式实现合并两个数组：

首先创建一个长度为两个数组长度之和的空数组，并且创建两个分别指向待排序数组第一个元素的指针；

```cpp
int p1 = 0; p2 = 0;  p = 0;
a[4],b[3],c[7];
```

现在依次比较两个指针指向的元素，如果指针一指向的元素小：
```cpp
c[p++] = a[p1++];
```

否则 ：
```cpp
c[p++] = b[p2++];
```

循环执行该过程，直到某一个数组的指针已经走到头了
```cpp
while(p1 < 4 && p2 < 3)
{
}
```

跳出循环后，将还剩下的元素直接放在辅助数组中即可
```cpp
while(p1 < 4)
{
	c[p++] = a[p1++];
}
while(p2 < 3)
{
	c[p++] = b[p2++];
}
```


对于本题，由于待排数组已经被扩大空间，所以可以使用更加快捷的方式，即原地合并，减小空间复杂度：

```cpp
int p1 = a.size() - 1;
int p2 = b.size() - 1;
int p = a.size() + b.size() - 1;

```
