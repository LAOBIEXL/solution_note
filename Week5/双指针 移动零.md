本质上是进行数组分块：

移动零的实现：

1. 使用备份数组，从第一个数组开始读取，读到非零元素就将其存下去，最后在最后几位补上零；

2. 不使用数组备份的办法：
双指针：

设定两个指针：
cur ：用来指向非零数和零数的分界，是最后一个非零数的下标；

i ：循环指针，用来指向遍历的位置

分块思路：
```cpp
cur = -1;
i = 0;
for(i = 0; i <= a.size() ; i++)
{
	if(a[i] != 0)
	{
		swap(a[++cur],a[i]);	
	}
}
```

注意：
```cpp
避免访问越界访问数组：

1遍历指针的自增在for循环内部；

int main()
{
	int cur = -1;
	int p;
	for (int p = 0; p < 15; p++)
	{
		if (a[p] != 0)
		{
			swap(a[p], a[++cur]);
		}
	}
	
}

2遍历指针的自增在循环内：

int main()
{
	int cur = -1;
	int p = 0;
	while (p < 15)
	{
		if (a[p] != 0)
		{
			swap(a[p++], a[++cur]);
		}
		else
		{
			p++;
		}
	}

	不要忘写else的p++;
}
```
