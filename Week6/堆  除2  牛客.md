### 思路：
将最大的偶数选出，并且除以2即可；

### 题解：
```cpp
#include<iostream>
#include<queue>
using namespace std;

typedef long long LL;

priority_queue<int> heap;

int main()
{
	LL n, k;
	LL sum = 0;
	cin >> n >> k;
	for (int i = 1; i <= n; i++)
	{
		LL t = 0;
		cin >> t;
		sum += t;
		if (t % 2 == 0)
		{
			heap.push(t);
		}
	}
	//堆的初始化成功；

	while (heap.size() && k--)
	{
		LL tem = heap.top() / 2;
		heap.pop();
		sum -= tem;
		if (tem % 2 == 0)
			heap.push(tem);
	}

	cout << sum << endl;

	return 0;
}
```

### 错误关键：
1. 对于除以2过的数据，它可能仍然是偶数，所以依旧需要放到堆中；
2. ==在进行k次操作的过程中，如果偶数的数量不足操作的数量，就会出错，所以每次使用容器时候一定要考虑到是否为空以及为空的后果！！==
